# Quer saber como instalar?

Instruções abaixo:
```sh
$ cd Backend-Game/
$ npm i
```
```sh
$ cd ../Frontend-Game
$ npm i
```
Após os módulos serem instalados, você vai precisar rodar o serviço front com express, nginx ou algo de sua preferência.
<br>
<br>
use: 
```sh
$ cd Frontend-Game && npm run build
```
para montar os arquivos HTML, CSS e JS
